# electron-gpio-raspberrypi
Electron.js Application for Raspberry Pi
### Installation


1. Clone the repo
```sh
git clone git@github.com:boraciner/electron-gpio.git
```
2. Install NPM packages
```sh
npm install
```
3. Rebuild modules
```sh
./node_modules/.bin/electron-rebuild
```
4. Run the application
```sh
npm run start
```
